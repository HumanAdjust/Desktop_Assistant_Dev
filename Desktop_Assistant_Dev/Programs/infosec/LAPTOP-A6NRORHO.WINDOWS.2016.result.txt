###################       6.24  IIS DB 연결 취약점 점검          ###################### 
####################################################################################### 
■ 기준 : .asp에 asp.dll 파일 맵핑 존재시(GET, HEAD, POST, TRACE) 양호                   
■      : 7.0이상- 요청 필터링에서 .asa, .asax 확장자가 False로 설정되어 있으면 양호     
                  요청 필터링에서 .asa, .asax 확장자가 True로 설정되어 있을 경우        
                  .asa, .asax 매핑이 등록되어 있어야 양호                               
                  (만약, Global.asa 파일을 사용하지 않는 경우 해당 사항 없음.)          
                 fileExtension=".asa" allowed="false" 이면 양호                    
                 fileExtension=".asa" allowed="true" 이면, .asa 맵핑 확인          
  	    : 6.0이하- .asa 매핑이 존재할 경우 양호
